Ce site web est hébergé par GitHub.

J'ai créé ce site web pour vous expliquer comment vous pouvez "facilement" héberger un site web GRATUITEMENT.

Pour cela, il vous suffit de créer un compte GitHub (si vous n'en avez pas déjà un) et de suivre les instructions disponibles sur ce site web.

1er vraie étape : Créer votre site web et le coder en html et css.

2ème étape : allez sur github.com.

NOTE : les menus de GitHub sont en Anglais.

3ème étape : cliquez sur la petite croix en Haut a droite de l'écran puis appuiez sur "create new repository".

4ème étape : Dans repository name , entrez le nom de votre site web. Par exemple : monsiteweb.github.io (Le nom doit impérativement se terminer par ".github.io" et le même nom ne peut pas être utilisé 2 fois).

5ème étape : Cliquez sur "Create repository".

6ème étape : Sur la nouvelle page, cliquez sur "uploading an existing file".

7ème étape : Glissez déposez vos fichiers html et css dans la zone prévue a cet effet.

8ème étape : Cliquez sur "Commit changes" en bas de la page.
